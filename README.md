# Neon Clash

Neon Clash is a modernized remake of the 2015 tutorial **“A Web Game in an Hour.”**  
This version updates the original browser game using modern HTML5, CSS3, JavaScript ES6+, and a responsive site layout.

## Features
- Modernized arcade paddle game
- Neon-themed interface
- Bounce-based gameplay
- New ball added after every 5 paddle returns
- First to 5 points wins
- Playable in the browser

## Pages
- `index.html` - project homepage
- `game.html` - playable game
- `devlog.html` - development log
- `technical-document.md` - technical explanation of the project

## Technologies Used
- HTML5
- CSS3
- JavaScript ES6+
- Canvas API
- GitHub
- GitHub Pages
